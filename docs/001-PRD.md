# Project Atlas

## Missão

Criar um micro SaaS de atendimento inteligente para pequenas empresas utilizando Inteligência Artificial.

---

## Objetivo

Desenvolver uma plataforma capaz de responder clientes automaticamente pelo WhatsApp, reduzindo o tempo de resposta e aumentando a conversão de vendas.

---

## Nicho inicial

Barbearias.

---

## Visão

O sistema deverá ser reutilizável para qualquer empresa apenas alterando suas configurações.

---

## Problema

Pequenos negócios perdem clientes porque demoram para responder mensagens.

Enquanto trabalham, não conseguem atender o WhatsApp.

---

## Solução

Criar um funcionário virtual capaz de:

- responder perguntas
- informar preços
- mostrar serviços
- informar localização
- informar horário
- registrar clientes
- encaminhar para atendimento humano

---

## Tecnologias

- n8n
- WhatsApp API
- OpenAI
- Hostinger VPS
- EasyPanel
- GitHub

---

## MVP

Versão 1

- Atendimento IA
- Base de conhecimento
- Registro de clientes
- Memória da conversa
- Transferência para humano

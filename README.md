# 🚀 Project Atlas

Micro SaaS de atendimento inteligente para pequenas empresas.

## Objetivo

O Atlas é uma plataforma de atendimento baseada em Inteligência Artificial.

O objetivo é automatizar o atendimento de empresas pelo WhatsApp, permitindo que pequenos negócios tenham um funcionário virtual disponível 24 horas por dia.

---

## Tecnologias

- n8n
- Supabase
- OpenAI
- WhatsApp API
- GitHub
